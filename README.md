# Python Voice Assistant (Online Version)
## Description
This project is a basic voice assistant developed in Python, created as part of my learning journey in the fundamentals of Artificial Intelligence and system automation.

The assistant uses speech recognition, text-to-speech, and Python automation to interact with the user, perform online searches, and execute basic system tasks.

This version mainly works online and requires an internet connection.

## Features
- Speech recognition (voice to text)
- Text-to-speech
- Tell the current time and date
- Online web search
- Play music and videos from YouTube
- Open applications
- Open files and folders
- Graphical interface using CustomTkinter


## Technologies and Libraries
- Python 3
- speech_recognition
- pyttsx3
- customtkinter
- webbrowser
- youtube-search
- pyaudio
- os, subprocess, datetime, urllib.parse

## Installation

Install dependencies

Bash

pip install speechrecognition pyttsx3 customtkinter pyaudio youtube-search
Note: PyAudio may require additional installation steps depending on your operating system.

## Usage
Run the application using:
python main.py
Speak into the microphone when the assistant is active.

The main goal of this project is to:
